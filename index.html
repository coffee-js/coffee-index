Vim 语法支持见 https://github.com/kchmck/vim-coffee-script

先访问 http://www.vim.org/scripts/script.php?script_id=3590 下载 zip 文件, 比如 vim-coffee-script-v001.zip , 注意版本可能不同, 然后执行(注意文件名) 

unzip -od ~/.vim vim-coffee-script-v001.zip

下一次打开 .coffee 文件就又高亮了.

Sublime Text 2 我用的免费版本, 测试如下: 具体看 https://github.com/jashkenas/coffee-script-tmbundle 以此执行以下命令, 新建 

git clone https://github.com/jashkenas/coffee-script-tmbundle.git 
cd coffee-script-tmbundle/ 
zip -r CoffeeScript.sublime-package *

最后把生成的文件转移到安装目录的"Pristine Packages", 具体可能不同: 

mv CoffeeScript.sublime-package ../Pristine Packages/
